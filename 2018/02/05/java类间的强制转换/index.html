子类和父类间的转换 或者说实现类和接口类之间的转换

java可以向上转型，就是说子类对象可以转换为父类型，转换后，该对象只能调用父类中拥有的方法和变量，而执行的却是子类中的方法体。看下

小结：有两种转换，一种是将子类对象转换为父类，另一种是将子类对象转换为父类后再强制转换为子类。这两种转换得到的对象最终都将只能调用父类中原有的方法，但是执行的方法体却是子类中的。接口类和实现类之间的关系也是这样，接口引用指向实现类对象（接口名 接口引用 = new 实现类名();）接口引用仅可以调用原接口声明的方法。
