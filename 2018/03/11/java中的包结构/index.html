java中的包结构

下载安装JDK之后，有个src压缩包，解压缩之后是这样的：

这里面的java文件夹大概就是我们开发的时候用到的，文件结构是这样的

1、java.applet：提供了需要创建一个小程序和用来跟其他小程序交流上下文的类。
2、java.awt：包含了所有用于创建用户界面和绘制图形和图像的类。
3、java.beans：包含了beans（基于JavaBean架构组件）开发相关的类。
4、java.io：提供了通过数据流、序列化和文件系统进行的系统系统输入和输出。
5、java.lang：java的核心类库，如基本数据类型、基本数学函数、字符串处理、线程、异常处理、系统缺省加载
6、java.math：提供用于执行任意精度整数算法 (BigInteger) 和任意精度小数算法 (BigDecimal) 的类
7、java.net：提供了实现网络应用程序的类。
8、java.nio：定义了缓冲器，它是数据容器，并且提供其他NIO包的概述。
9、java.rmi：提供了RMI包。
10、java.security：提供了安全框架的类和接口。
11、java.sql：提供了使用Java编程语言访问和处理存储在一个数据源（通常是一个关系数据库）的API。
12、java.text：提供了语言无关的方式处理文本、日期、数字和信息的类和接口。
13、java.time：日期、时间、时刻和时间段的主要API。
14、java.util：包含了集合框架、遗留的集合类、事件模型、日期和时间工具、国际化和各种各样的工具类（一个字符编译器、一个随机数生成器和一个位数组）。
参考自http://blog.csdn.net/lc0817/article/details/54773516
在这些包中还有包，java的类库很多
学习一下常用包吧

java.lang

lang包中，有我们的基本类型包装类(Integer.java等)、Class.java、Comparable.java、Cloneable.java、Exception.java、Error.java、Math.java、Object.java、Override.java、Package.java、Runnable.java、Runtime.java、String.java、StringBuffer.java、StringBuilder.java、System.java、Thread.java、Iterable.java还有各种Exception的子类等等等等。核心类库，一点也不为过。
这里面还有
java.lang.annotation包，这个包里面是java的注解，有Annotation.java ElementType.java Target.java等等
java.lang.reflect包，提供了用于获取类和对象的反射信息的类和接口。反射是java的一大特性，一定要明白才行。
