java中的volatile关键字

作用：
1、volatile修饰的变量不会被某个线程内部缓存和重排序，也就是说他修饰的变量被改变时，是直接操作的内存，这样保证了他修饰的变量具有可见性。
2、volatile修饰的变量参与地指令将不会被重排序，保证了线程之间操作的有序性。

什么是可见性？
可见性是指多个线程之间的关系，当一个线程修改线程共享变量后，内存会立即得到修改，其余线程再操作这个变量时，能得到最新的数据，也就是一个线程修改了一个数据，其他线程可见，这个叫做可见性。Java内存模型是这样的，当一个线程修改某个变量后，会将新值同步回主内存，线程读取前从主内存刷新变量值。
某个线程对volatile修饰的变量的保证了新值能立即同步到主内存中，当线程读取的时候，也会立即从内存中刷新，所以，所有线程访问的时候，都会是得到最新的数据，达到了线程之间的可见性。Java中除了volatile还有synchronized和final也实现可见性。

什么是有序性？
有人总结为“在本线程内观察，所有操作都是有序的，在一个线程看另一个线程，所有操作都是无序的”，java提供了volatile和synchronized关键字来保证线程之间操作的有序性。volatile关键字表示了禁止指令重排序的语义，而synchronized表示了持有同一个锁的两个同步块只能串行的执行

java多线程中除了可见性、有序性、还有个原子性
什么是原子性？
原子性是指，一个线程中的某个操作，一旦被cpu执行，便不可中途暂停然后继续完成。比如a++这个操作，就不是原子性的。Java中可用关键字Synchronized让非原子操作变为原子操作。
