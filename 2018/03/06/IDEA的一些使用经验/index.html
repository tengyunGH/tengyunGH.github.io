IDEA

IDEA的一些使用经验

一些便捷的配置

1、settings -> Editor -> File and Code Templates 里面可以设置创建一个类的时候默认的文档注释

2、file->other settings->default settings 在这里配置一些所有项目的设置，比如maven的配置

3、安装的bin目录下的idea.properties这里面修改IDEA的系统文件位置 从c盘改到其他的盘

编译方式的讲解

编译方式有三种，build Recompile Rebuild
1、build project 对选中的project，有改变的进行编译，没有改变的不编译（make）
build module 对选中的module，改变的进行编译

2、Recompile 对选中的java类进行重新编译（无论是否重新修改过）

3、Rebuild 对选中的project进行全部重编译，没改过的也重新编译（是个比较浩大的工程，用的较少）

这三种情况，针对的目标是：Rebuild的对象只能是project，Recompile的对象只能是Java类文件，Build的对象是project和module，其中build用的最多，只编译改变的文件

一些目前常用的快捷键

ctrl + shift + alt + s 调出项目配置
ctrl + alt + s 调出配置
ctrl + shift + A 调出查询某操作快捷键

ctrl + alt + l 格式化代码
alt + E 调出run configuration
实际上，快捷键是可以在 settings->keymap中查询或者修改，你可以选用你自己熟悉的，然后多用用就好了
